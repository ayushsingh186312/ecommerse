## Run Locally

Clone the project

```bash
  git clone https://github.com/followDev/ecommerce-react.git
```

Go to the project directory

```bash
  cd ecommerce-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
